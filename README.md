Stranger Listening Platform 👂✨

A safe space for 30-minute, anonymous conversations — no pressure, no judgment, no advice unless requested.

**Live Site**: [https://silent-stranger-sessions.vercel.app](https://silent-stranger-sessions.vercel.app)

---

## 🛠️ Tech Stack

This project is built with:

- [Vite](https://vitejs.dev/) – blazing fast dev environment
- [React](https://react.dev/)
- [TypeScript](https://www.typescriptlang.org/)
- [Tailwind CSS](https://tailwindcss.com/) – modern utility-first styling
- [shadcn/ui](https://ui.shadcn.com/) – beautiful component library
- [React Router](https://reactrouter.com/) – routing
- [Calendly Integration](https://calendly.com/) – for session booking

## 🚀 Getting Started Locally

To run this project locally:

### Prerequisites

- Node.js & npm  
  Recommended: use [nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

### Installation

```bash
# Clone the repository
git clone https://github.com/donganh1409/silent-stranger-sessions.git

# Navigate into the project
cd silent-stranger-sessions

# Install dependencies
npm install

# Start the local development server
npm run dev

Your site will be live at http://localhost:5173

🌍 Deployment

This project is deployed via Vercel.
Every push to the main branch is automatically deployed to:

🔗 https://silent-stranger-sessions.vercel.app

✅ Want to connect a custom domain?
	1.	Go to Vercel Dashboard
	2.	Open your project
	3.	Navigate to Settings > Domains
	4.	Add your custom domain (e.g., strangertalk.app)

Docs: Vercel Custom Domains
